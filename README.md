# ShootingProject (타워 디펜스 - 미사일 막기)

Developed with Unreal Engine 5

Creating by
1 month of Unreal Engine
2 people 2 weeks project
Using Blueprint

- 어느날 전쟁이 일어났다.
나는 우리나라 특공대 707 부대원.
상부에서 명령이 떨어 졌다.
“마지막 거점을 사수하라”
남은 건 내 두 몸과 총알 뿐
몰려드는 탱크와 전투기의 미사일을 방어하라
